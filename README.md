# Desafio Abs Api

## Configuração

Para executar o projeto, será necessário instalar os seguintes programas:

- JDK 17: Necessário para executar o projeto Java
- Maven 3.8.7: Necessário para realizar o build do projeto Java
- Ides Spring Tools 4 ou IntelliJ: Para o desenvolvimento do projeto

## Teste da Documentação

Por fim, basta acessar a url: http://127.0.0.1:8080/swagger-ui/index.html#/ na máquina que esteja executando o projeto, Terá a documentação da api pelo Swagger.
